All Examples

Examples are checked into the github repository. You can take a look with:

> git clone git://github.com/firebase/firepad.git
> open firepad/examples/all-examples.html

Or you can view them below and click "View Source on Github" to jump to the source.

1. Rich Text

Use Firepad's rich text features for collaborative blogging, note taking, proofreading and much more!

2. Code Editing

Combined with CodeMirror's extensive code editing features, Firepad is great for pair programming!

3. User List

You can easily add user presence information and other features.
